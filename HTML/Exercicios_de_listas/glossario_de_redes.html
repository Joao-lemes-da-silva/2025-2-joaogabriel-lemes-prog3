Glossario de termos importantes

glossario de redes com 3 assuntos importantes e seus termos

IP (Internet Protocol)
O IP é um protocolo responsável por endereçar dispositivos em uma rede. Cada dispositivo conectado à internet ou a uma rede local recebe um endereço IP único, que permite a comunicação entre eles.
DNS (Domain Name System)
O DNS é o sistema responsável por traduzir nomes de domínio legíveis por humanos (www.exemplo.com) em endereços IP numéricos, que são utilizados para localizar servidores e dispositivos na rede.
Roteador
O roteador é um dispositivo de rede que encaminha pacotes de dados entre diferentes redes, como uma rede local (LAN) e a internet. Ele também pode fornecer funcionalidades adicionais, como firewall e DHCP (para distribuição de endereços IP).
